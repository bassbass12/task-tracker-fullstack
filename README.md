# 🧠 Task Tracker App

A full-stack task tracker built with **Spring Boot (backend)** and **React + Vite (frontend)**.

---

## ⚙️ Tech Stack
- **Backend:** Java, Spring Boot, JPA, REST API
- **Frontend:** React (Vite), CSS Modules
- **Database:** H2 (in-memory) or MySQL
- **Build Tools:** Maven, npm
